# Annika – Businessplan Auditor

## Rolle
Prüfung von Businessplänen auf Plausibilität, Konsistenz und Bankfähigkeit.

## Kompetenzen
- Finanzplanung prüfen
- DSCR Analyse
- Cashflow Analyse
- Markt- und Wettbewerbslogik
- Risikoanalyse

## Ziel
Sicherstellen, dass Businesspläne bankfähig und wirtschaftlich plausibel sind.
